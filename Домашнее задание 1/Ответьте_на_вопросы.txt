1.В чем разница между папками /bin и /sbin в корневом каталоге ОС Linux
/bin содержит основные команды, доступные всем пользователям, /sbin - системные утилиты для администраторов.

2.Что будет, если создать файл или папку в ОС Linux с одинаковым названием,
        но с разным регистром? (например Folder и FOLDER )
Linux различает регистры, поэтому "Folder" и "FOLDER" будут разными объектами.

3.Назовите временную папку в  ОС Linux и Windows
Временные папки: /tmp (Linux), C:\Windows\Temp (Windows).

4.Почему версий ОС Linux существует такое большое количество?
Много версий Linux из-за открытого исходного кода, позволяющего адаптировать ОС для разных целей.

5.Для чего нужна папка /opt, расположенная в корневом каталоге
/opt используется для установки сторонних приложений.

6.Для чего нужна командная оболочка?
Командная оболочка - интерфейс для ввода команд и взаимодействия с ОС.

7.Что такое ядро ОС?
Ядро - это основная часть ОС, управляющая аппаратными ресурсами.

8.Назовите несколько ОС под управлением ядра Linux
Примеры ОС на ядре Linux: Ubuntu, Debian, Fedora.
